# The Agents - Visual Assets Brief

## Brand Guidelines
- **Colors:** 
  - Primary: Deep Blue (#1a237e)
  - Secondary: Tech Green (#4caf50) 
  - Accent: Gold (#ffc107)
- **Font:** Inter (Google Fonts) - Mix of Thin + Bold
- **Style:** Modern, clean, tech-forward, professional but approachable
- **Logo:** theagents.wtf brand (minimalist)

## Post Designs Required

### Post 1: "5 tareas mientras dormís"
- Split screen: sleeping person vs working dashboard
- 5 carousel slides with icons
- Dark theme (night vibes)
- Icons: email, chart, calendar, report, spy glass

### Post 2: "¿Cuánto cuesta NO tener IA?"
- Calculator theme
- Red vs Green comparison
- Big bold numbers
- ROI visualization

### Post 3: "Agente vs Empleado"
- Boxing ring aesthetic
- VS layout
- Stats comparison
- Human icon vs Robot icon

### Post 4: "3 errores con IA"
- Traffic light design
- Warning icons vs Checkmarks
- Clean, educational layout
- Process flow elements

### Post 5: "Competencia ya se movió"
- Spy/intelligence theme
- Dark dashboard
- Charts and data visualizations
- Urgent but not aggressive

## Specifications
- **Size:** 1080x1080 (Instagram square)
- **Format:** PNG/JPG for static, MP4 for motion
- **Brand consistency:** Logo placement bottom right
- **Typography hierarchy:** Clear, scannable
- **Mobile-first:** Readable on small screens
